# persistent pip_packages directory on load (the same mechanism feedback-ultrastar-import uses for
# numpy). Every one is imported LAZILY and only when its source type is actually used, so the core
# plugin and the dependency-free types (Remote Library Server, Google Drive) still run on a deploy

#                         key-stretch + OpenPGP decryption of the E2EE key hierarchy + blocks.
#   iroh                — the Remote Server (iroh) type (iroh-library.v1): a peer-to-peer QUIC
#                         transport that tunnels the Remote Library Server protocol, so a
#                         home-hosted server is reachable by its Library ID with no port forwarding.
